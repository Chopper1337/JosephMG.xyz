Cartesian Product

The Cartesian Product of two sets is the set of all ordered pairs between the two sets.

Example: X = {a,b,c} and Y = {w,z}

X*Y = The Cartesian Product, which is the following

X*Y={(a,w),(a,z),(b,w),(b,z),(c,w),(c,z)}

Relation

A Relation between sets X and Y is any subset of the Cartesian Product X*Y

Example: Referring to the X and Y sets from the previous example, we can define realtions R, S and T as

R={(a,w),(b,w),(c,z)}

S={(a,w),(b,w)}

T={(a,w),(c,w),(c,z)}

Domain

A Domain of a Relation is the set of the first components of the Realtion

Example: If R is a Relation from set X to set Y

Then Domain R = {x|(x,y),E R}

Example: Looking at R, S and T in the previous example

Domain R = {a,b,c}

Domain S = {a,b}

Domain T = {a,c,z}

Range of a Relation

The Range of a Relation is the set of the second components of the pairings in the Relation. If a value repeats, you don't include it twice.

Range R = {y|(x,y),E R}

Range R = {w,z}

Range S = {w}

Range T = {w,z}

Domain Restriction (Left Restriction)

In general, given a Relation R from sets X to Y

If T is a subset of the set X: T (Left Restrcition) R = {(x,y)|x E T and (x,y) E R}

i.e All pairings in R whose first component lies in T.

Domain Anti-Restriction

If T is a subset of X: T (Anti-Restriction) X = {(x,y) | !E T and (x,y) E R}

Range Restriction

If w is a subset of Y, then R restricted to w = {(x,y) | y E W and (x,y) E R}

i.e All pairings in R whose second component lies in W

Range Anti-Restriction

If w is a subset of Y, then R anti-restricted to w = {(x,y) | y !E w and (x,y) E R}

Forward Composition

We can combine two relations by using forward composition. The symbol is ';'

If R and S are relations, we do R first and then S

Note: In forward composition, the final ordered pairing consists of the starting value and the end value. There is a link that connects those two values.

If R is a relation that maps to set X to set Y and S is a relation that maps set Y to set Z

Then R;S maps set X to set Z

R;S = {(x,y) | there exists y E Y such that (x,y) E R and (x,z) E S}
